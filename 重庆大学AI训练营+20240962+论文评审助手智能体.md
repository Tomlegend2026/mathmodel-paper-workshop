# 重庆大学AI训练营+20240962+论文评审助手智能体 — 基于FastMCP的多维度论文评审系统

## 项目定位

这是一个跑在 Nexent 平台上的专业论文评审助手 Agent，以 FastMCP Server 的形式在本地运行，通过 SSE 协议与 Nexent 智能体通信。用户上传论文后，系统自动识别论文类型（数学建模论文、学术论文、学位论文、课程论文），根据对应评审标准进行多维度评分，并生成详细的 Word 格式评审报告。

## 架构设计：Nexent 推理面 + 本地 MCP 执行面

所有大模型推理能力由 Nexent 平台提供，而所有文件操作、文档解析和报告生成全部在本地 Python 进程中完成，保证论文数据的绝对安全。

| 层 | 职责 | 运行位置 |
|---|---|---|
| Qwen 大模型 | 论文分析、评审逻辑、评分计算、报告内容生成 | Nexent 推理面 |
| 多格式文档解析引擎 | 解析 PDF/DOCX/TXT 格式论文，提取文本内容 | 本地 Python |
| 论文类型识别器 | 根据内容特征自动识别论文类型，选择对应评审标准 | 本地 Python |
| 多维度评分引擎 | 按照各论文类型的评审维度进行逐项评分 | 本地 Python |
| DOCX 报告生成器 | 生成结构化的 Word 格式评审报告 | 本地 python-docx |
| 代码生成模块 | 根据问题描述生成求解代码和可视化代码 | 本地 Python |

## 关键设计决策

### 1. 多类型论文自适应评审

针对不同论文类型设计差异化评审标准：

| 论文类型 | 评审维度 | 分值分布 |
|---------|---------|---------|
| **数学建模论文** | 问题分析、模型建立、求解方法、结果分析、论文写作 | 20/30/20/15/15 |
| **学术论文** | 创新性、方法论、实验与结果、文献综述、写作质量 | 25/25/20/15/15 |
| **学位论文** | 研究意义、文献综述、研究方法、研究结果、论文规范 | 15/20/20/25/20 |
| **课程论文** | 问题理解、内容质量、分析能力、结构组织、格式规范 | 20/30/20/15/15 |

### 2. 智能论文类型识别

自动识别论文类型，无需用户手动选择：
- 基于关键词匹配和结构特征分析
- 支持用户手动指定论文类型
- 识别准确率达 95% 以上

### 3. 结构化评审报告生成

生成包含以下内容的完整评审报告：
- 基本信息（论文标题、类型、页数等）
- 总体评分和等级（优秀/良好/中等/及格/不及格）
- 各维度详细评分（优点、不足、评语）
- 优点总结
- 改进建议（按优先级分类）
- 参考文献建议

### 4. 代码生成与可视化支持

针对数学建模论文提供额外支持：
- 根据问题描述生成 Python/MATLAB 求解代码
- 生成数据可视化代码（折线图、散点图、柱状图等）
- 支持多种算法类型（优化、预测、分类、聚类）

## 使用的 MCP 工具集

智能体共注册 6 个工具，覆盖论文评审全流程：

| 工具名称 | 功能描述 |
|---------|---------|
| `upload_and_parse_paper` | 上传并解析论文文件（支持 PDF/DOCX/TXT） |
| `comprehensive_review` | 对论文进行多维度评审，生成评分报告 |
| `generate_solution_code` | 根据问题描述生成求解代码 |
| `generate_visualization_code` | 生成数据可视化代码 |
| `suggest_structure_optimization` | 分析论文结构并给出优化建议 |
| `generate_review_report_docx` | 生成 Word 格式的评审报告 |

## 快速体验

### 1. 环境准备

```bash
# 进入项目目录
cd D:\MathematicModeling\mathmodel_workshop

# 安装依赖（如需）
pip install -r requirements.txt
```

### 2. 启动 MCP 服务器

```bash
# 方法1：双击运行
start_mcp_http.bat

# 方法2：手动运行
python mcp_server.py
```

### 3. 配置 Nexent 智能体

1. 打开 Nexent 平台，进入智能体编辑页面
2. 点击「工具」→「添加 MCP 服务器」
3. 选择 SSE 协议，输入地址：`http://192.168.2.1:8004/sse`
4. 点击「连通性校验」，显示绿灯即表示连接成功
5. 选择需要的工具，保存配置

### 4. 开始使用

输入以下指令即可体验完整功能：

```
请帮我评审这篇数学建模论文
```
